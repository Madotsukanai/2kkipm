ゆめ2っき用非公式パッケージマネージャーです。 / An unofficial package manager for Yume 2kki.

releaseのバイナリはWindows版とLinux版しかありませんが、macOS等でもビルドすれば使えると思います。 / Release binaries are only provided for Windows and Linux, but it should work on macOS and others if built from source.

## 使い方 / Usage
- **update** - Wikiからパッケージリストを更新し、最近の更新一覧を表示する / Update the package list from the Wiki and show recent updates
- **upgrade** - 最新バージョン情報を表示・アップデートする / Show latest version info and update the package manager itself
- **install** -  coreまたはpatchをインストールする / Install a core or patch
- **list** - インストール履歴を表示する / Show installation history
- **show** - 最近の更新一覧を表示する / Show the list of recent updates
- **config** - 設定を表示・編集する / View or edit configurations
- **clean** - 状態ファイルをリセットする / Reset the state file and clear cache
- **remove** - インストール済みのバージョンを削除する / Remove an installed version
- **help** - ヘルプを表示する / Print this message or the help of the given subcommand(s)
