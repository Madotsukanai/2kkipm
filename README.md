# 2kkipm
ゆめ2っき用非公式パッケージマネージャーです。releaseのバイナリはWindows版とLinux版しかありませんが、macOS等でもビルドすれば使えると思います。
## 使い方
- **update** - Wikiからパッケージリストを更新し、最近の更新一覧を表示する
- **upgrade** - 最新バージョン情報を表示・アップデートする
- **install** -  coreまたはpatchをインストールする
- **list** - インストール履歴を表示する
- **show** - 最近の更新一覧を表示する
- **config** - 設定を表示・編集する
- **clean** - 状態ファイルをリセットする
- **remove** - インストール済みのバージョンを削除する
- **help** - Print this message or the help of the given subcommand(s)
